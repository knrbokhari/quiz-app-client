## Install

    using SSH:
    $ git clone git@github.com:knrbokhari/quiz-app-client.git
    or using HTTPS:
    $ git clone https://github.com/knrbokhari/quiz-app-client.git
    $ cd quiz-app-client
    $ npm install

### Run the next dev

    $ npm run dev

### Run the build

    $ npm run build

### Run start

    $ npm start
